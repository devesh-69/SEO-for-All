## Why this repository exists

Most SEO resources explain *what to do*.

Very few explain **why it works**.

This repository is designed to visually demonstrate how search engines read:
- HTML structure
- internal links
- indexing signals
- performance metrics
- JavaScript-rendered content

Using clean HTML, CSS, and controlled examples.
